JavaScript基本语法
2020-06-25|Web开发

上一章节介绍了js的一些旧历史，这次就介绍一下基本语法吧，和js来一场美丽的邂逅，哈哈哈…

初识使用JavaScript

上面我写了一个html页面代码<script>标签中存放的就是js代码

以下是需要注意的几个要点

  1. 注释符
    1. //单行注释
    2. /*注释内容*/,多行注释
  2. ECMA中的语句以一个分号结尾。但是可以省略，如果省略了由于浏览器内的解析器自己确定语句
    的结尾。
  3. 输出内容的小工具：
    1. alert() ，以弹出框的形式输出内容。
    2. console.log() ，在控制台内以日志形式打印内容。
    3. document.write() ，将内容显示在页面上。
  4. javaScipt中的一切都严格区分大小写(html对大小写不敏感)
  5. javaScript要写的地方？
    有行内写法也有内嵌写法(一般使用内嵌写法)

变量的定义

  1. 先定义变量在赋值
  2. 定一变量时直接赋值

变量定义规则

  1. 数字、字母、下划线、 $符 。
  2. 不能以 数字 开头。
  3. 不能和关键字、保留字相同。

变量的类型

  • 数据类型（存储的数据的类型）
    基本数据类型和对象（引用类型）。
  • 基本数据类型
    • 数字 Number
    • 字符串 String
    • 布尔 Boolean
    • undefined
    • null
  • 对象数据类型（后续说）

  1. Number类型

    一旦超过了最小值和最大值的范围，这个数值将会被转换为 -Infinity 或 Infinity 。
    小数在进行计算时会出现一些问题，这是每个语言几乎都会出现的

  2. String 字符串
    由单引号或双引号围起来，这两者没什么区别
    引号嵌套问题需要进行转义字符的帮忙

  3. Boolean类型
    布尔值，只有两个结果：true（真）或false（假）

  4. undefiend类型
    只有一个值，是undefined,代表着变量未初始化

  5. null类型
    只有一个值，是null，有些人可能问了，这个为什么是个object类型，其实是在一开始定义规则的人员就将此定义错了，之后便将错就错,如果进行大规模的更改，势必会给很多公司重构带来很多麻烦

  • 获得变量的类型
    typeof 值/变量，最终得到的结果数据类型名的小写形式。
    返回值：boolean、string、number、undefined(如果变量定义都未定义是无法使用的，会报错，这种方法除外)、object
